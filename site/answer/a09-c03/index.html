第九幕·参考答案：所有动作都重要时，哪一个应该先让步

练习 1

先淘汰 A，因为它已经侵入足底保留边界。B 与 C 都满足 5 mm/1.0° 的手部许可并保留足底余量，才能进入严格层级比较；若 C 只放宽最低层姿态且没有恶化上层最优残差，应优先比较它与 B 在既定层级中的残差和松弛，而不是比较无语义总分。

A 违反的是硬可行性，B 与 C 的差别才属于词典序任务与有限松弛。总代价最小的 A 看起来最漂亮，实际上它用手部 1 mm 的精度换掉了足底边界。

还要逐项回代压力中心、手部位置与姿态、关节限位和每层最优残差。本题只支持这一步筛选，不证明 B 或 C 能在真实身体上执行。

练习 2

不能称为严格优先。应把足底单边、摩擦和 CoP 等不可交换条件放入硬约束；对于必须严格先满足的任务，使用逐层冻结上层最优值的 HQP，并为不同量纲声明尺度和容差。

这是一个加权求和的尺度反例。有限权重始终允许交换，单位变化还会改变残差数值。若继续把权重放大，矩阵可能先变得病态，而偶然得到的解仍不能充当层级保证。

可以分别用米和毫米表达同一物理任务：若正确归一和词典序实现下物理解仍改变，就存在语义或数值问题。本题不证明某个分层二次规划软件实现正确。

练习 3

不能发送当前结果。先缩小或撤销低层动作，恢复上层残差；随后按新的数值秩、阻尼和关节限位重新构造层级问题。若无法保持上层结果，就保留高层任务并放弃左手接近。

博士把这种现象称为零空间投影泄漏。近奇异时，阻尼投影通常只近似满足 A1N1=0A_1N_1=0A1​N1​=0。低层误差变小会诱使我继续执行，但它不能抵消上层足底任务已经退化的事实。

应直接计算 A1N1ΔzA_1N_1\Delta zA1​N1​Δz 并回代上层残差，而不是只看投影器名称。证据只覆盖当前线性化与固定容差。
