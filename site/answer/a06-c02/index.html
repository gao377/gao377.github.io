返回题目

这四题先判断记录能够排除什么，再决定是否更新模型。一次吻合只能支持局部一致性；参数更新还需要可辨识性和独立姿态证据。

练习 1

我应保留当前模型，先把这组结果登记为“该姿态下的负载传播量级一致”，不能直接更新工具质量。逐项残差为

[0.1, 0.1, −0.1, 0, 0, 0, 0]T N m,[0.1,\ 0.1,\ -0.1,\ 0,\ 0,\ 0,\ 0]^{\mathsf T}\ \mathrm{N\,m},[0.1, 0.1, −0.1, 0, 0, 0, 0]T Nm,

它说明测量与预测接近，却只有一个姿态、一个负载方向。工具质量、质心位置、关节偏置和腕部外力映射仍可能在此处互相抵消。

博士把“相同结果可由多组参数造成”的问题称为参数相关性。下一项应在安全范围内改变手臂姿态和工具朝向，并保留相同的 30 N30\ \mathrm N30 N 受控转移；若差值随 JTFJ^{\mathsf T}FJTF 和重力方向持续变化，工具负载解释才获得更多支持。另可用独立秤量或吊架记录检查转移量。

最容易犯的错是把小残差等同于参数正确。若把腕部外力和工具负载同时完整加入模型，还会把同一重量计算两次。复核时必须明确传感器位于系统边界的哪一侧。当前记录不授权夹爪独立承载毛坯。

练习 2

我应先检查摩擦模型和速度记录，不增大连杆质量。残差随速度方向近似翻转，停住又接近零，更像 Coulomb 摩擦项

τcsgn⁡(q˙)\tau_c\operatorname{sgn}(\dot q)τc​sgn(q˙​)

没有被正确描述。博士把这种由运动方向留下的模式称为摩擦残差纹理。

下一轮应保存速度符号、速度大小、温度、换向附近的停留时间和其他关节状态；用多种非零速率观察是否还叠加与速度大小成比例的黏性项。零速附近不要强行套用符号函数，因为静摩擦可能取一个区间。

若误把它归因于质量，模型会在不同重力姿态产生错误补偿，而且正反向残差仍不能同时消失。我会在相同姿态下改变速度方向与大小，再去不同姿态的静止点检查重力项。当前数字只能支持优先检查摩擦，不能唯一确定摩擦参数。

练习 3

我应保留关节动力学参数不变，把问题先放在腕部测量链。重新校偏只改变腕部法向记录，而关节作用残差完全不变，说明这次操作修正的是力觉零点偏置，不是机械臂惯量或关节作用偏置。

腕部校偏不能直接修正关节模型。若动力学计算使用腕部外力作为输入，校偏后的数值会间接改变 JTFextJ^{\mathsf T}F_{\mathrm{ext}}JTFext​，但这仍是输入修正，不是把关节惯量改掉。

两条记录即使都以牛顿或牛顿米出现，也不能把偏差混到同一参数里。我会先在无接触静止窗口复查腕部偏置与工具重力补偿，再用关节侧静止重力姿态检查关节残差。当前现象不排除其他关节模型误差，只说明这 3.2 N3.2\ \mathrm N3.2 N 不应拿来改关节参数。

练习 4

我应回退旧模型，并设计新的安全激励；不能直接发布四个参数，也不能简单发布前三个原始参数。按阈值

18.0, 3.7, 0.8>0.02,0.006<0.02,18.0,\ 3.7,\ 0.8>0.02, \qquad 0.006<0.02,18.0, 3.7, 0.8>0.02,0.006<0.02,

缩放后的数值秩为 3。博士把几乎看不见的组合称为不可辨方向。它通常由四个原始参数共同组成，所以“前三个奇异方向可辨”不等于“前三个原始参数可辨”。

更关键的是，新参数让独立姿态残差增加 35%35\%35%。这说明它只改善了参与估计的数据，尚不能成为新的身体模型。可在限位、速度、作用和支撑边界内改变关节组合，使最弱方向对观测产生不同影响；也可用独立测量固定其中一个参数。

拟合集残差减半还不足以发布结果。我还要重新计算缩放、奇异向量、参数物理可行性和未参与估计的姿态残差。若补充激励仍不能提高最弱方向，就只登记可辨的基参数组合与未知范围，而不是制造四个小数。
