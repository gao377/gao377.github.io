路留下的形状——从四足运输到轮足道路运动

全书路线

我仍背着走出受损通道时的那件毛坯。长平路让我发现，四足身体即使能穿过坏路，也会在远距离运输中反复抬落、消耗时间和能量。于是博士没有换掉我，而是在同一具身体上增加轮端；从那一刻起，旧状态、旧地图和旧动作都要按改变后的身体重新理解。

这一幕里，我学会滚动，也知道何时迈步。滚动较省力，不代表轮子愿意横移、转弯或在湿地上一直抓住地面；迈步能够越过局部障碍，也不代表每次卸载和触地都成立。当两种方式都不能留在各自边界内，停下来、退回或求助同样是有效的结果。

第一次顺着路滚起来

  1. 平路为什么也会让我疲惫
  2. 身体换了一部分，旧经验还能用吗
  3. 轮子为什么只愿意沿一个方向走
  4. 我数出的距离可信吗

这里先回答轮端怎样成为身体的一部分，以及编码器、轮半径和滚动约束怎样把轮子的转动连接到机身运动。直路能走通以后，弯道、载荷和障碍才开始暴露这种新身体的代价。

路并不总肯让轮子过去

  1. 路向左弯，为什么四个轮子在互相拉扯
  2. 轮子说我到了，墙角为什么还没到
  3. 斜路为什么把货物推向一边
  4. 一根矮管为什么能拦住我

固定轮的转弯冲突会进入里程推算，拱形路面会改变各轮承载，低管线则迫使我在滚动与抬轮之间改变接触。每一次改变都要保留卸载、支撑、净空、触地和停止条件。

当路面不再可靠

  1. 轮子越转越快，我为什么反而没有前进
  2. 地面还能借给我多少力
  3. 继续滚、抬腿，还是停下来
  4. 我已经到了，为什么修理仍未开始

湿路把“轮子正在转”和“身体正在前进”分开，也让我只能从有限观察中保守判断地面还能提供多少作用。最后的路线决定先排除越过硬边界的候选，再比较能耗、时间与切换代价；抵达旧工坊以后，途中发生过的故障与未知仍被保留，而毛坯依然固定在背上。
