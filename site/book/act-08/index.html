第八幕

门按人的尺度打开——从人形重构到传统双足控制

全书路线

我已经找到了泵站。地图没有骗我，路也确实把我带到了这里。可门洞比我的身体更窄，台阶沿着人的步幅向上延伸。我第一次明白，到达一个地方，并不等于已经拥有进入它的身体。

博士没有把这次重构说成一次“升级”。四足身体让我跨过碎石，轮足身体让我沿城市远行，机械臂让我制作并携带维修部件；它们解决过的真实问题不会因为我改成人形而变得低级或错误。现在改变身体，只因为八级台阶、狭窄平台、门把手和泵站面板共同形成了另一组边界。

记忆、任务、地图、维修痕迹和背上的部件仍由同一个“我”承担，但旧身体的质量、坐标、碰撞外形、状态估计和控制参数不能直接搬进新身体。排水渠巡检设备与城市保障运输设备也保持原来的形态和任务：只有我被重新组织，它们不会因为我的身体重构而突然长出人的双腿。

这一幕先让新身体知道自己是谁，再让双脚真正承担有限接触。欠驱动、冲击、LIPM、ZMP、捕获、角动量与周期稳定，都在我发现“关节听话仍会跌倒”以后进入。最后，落脚区域、脚步图、混合整数规划、全身碰撞和事件重规划把理论带回八级台阶与窄门。这里建立的是传统双足控制基线；本幕不使用强化学习解决双足失败，也不让学习方法替我跳过模型与复核。

身体改变以后，哪些东西仍属于我

  1. 门洞只窄四厘米，为什么要改变整具身体
  2. 身体换过以后，哪些东西仍然属于我
  3. 只弯一条腿，身体为什么会跟着倾斜
  4. 我还记得整座城，为什么不知道自己怎样站着

重构先区分身体适配与形态等级，再用跨身体谱系保存记忆和责任，却不伪造参数连续。标定、质心与惯量、浮动基状态估计随后按失败出现：关节编码器只告诉我身体内部怎样弯曲，可靠接触和惯性观测才可能帮助我理解整个身体相对世界怎样运动。

双脚都在地上，为什么平衡仍然会失去

  1. 每个关节都听话，我为什么还是会倒下
  2. 脚没有移动，支撑为什么还会变化
  3. 脚落地的一瞬间，我的速度去了哪里
  4. 博士为什么把我画成一根倒立的摆
  5. 脚下的点还在范围里，我为什么停不下来

输出约束揭示“关节任务完成”以后仍会留下浮动基内部运动，有限足底又把接触力与力矩限制在可实现的区域。触地会让速度瞬间改变，连续动力学与冲击重置因此组成混杂系统。LIPM 和 ZMP/CoP 是带前提的简化，不是把人形永久缩成一个点；它们能解释一部分质心运动，也明确暴露自己忽略的角动量、高度变化与足底滚转。

身体已经偏离时，怎样判断还来不来得及

  1. 下一只脚落在哪里，我才不用继续追赶自己
  2. 我该用脚踝、转动身体，还是再迈一步
  3. 前十步都相似，第十一步为什么开始摇晃

捕获点与 DCM 把“现在的速度会把我带到哪里”写进下一步，恢复决策再依次比较脚踝、角动量和迈步的真实余量。周期行走不能只凭十步没有跌倒来证明；Poincaré 返回映射和 Floquet 乘子在同一触地相位观察误差是否逐步缩小，也保留模型、事件与局部结论的边界。

一串可落脚的位置，怎样变成一次有记录的进入

  1. 脚能够碰到台阶，为什么还不一定能踩上去
  2. 八级台阶，为什么不能把同一步复制八次
  3. 脚步计划通过了，膝盖和背上的部件呢
  4. 计划还没有算完，下一步要不要迈

足底中心可达不等于整只脚能够承重，单步安全也不等于八步相连。鲁棒落脚区域先扣除脚掌尺寸与不确定性；脚步图、A* 和混合整数规划再共同安排左右脚、区域、位置与时机。离散脚点最终必须展开为脚尖、脚跟、膝、骨盆、双臂、传感器和背部部件的连续轨迹，并通过触地冲击与接触确认。

第五级被水打湿以后，旧计划不会因曾经通过就永久有效。表面年龄、当前接触阶段、当前最好可行解和后备动作决定下一步是迈出、等待、退回还是停止。只有八级接触、湿润事件、平台转身、窄门扫掠与门内稳定停驻都留下闭合记录，我才有资格说“进入”。

重新组织成人形没有抹去四足和轮足留下的经验，也没有把它们降成较低的阶段。它只是让我第一次能够沿着这座城市按人类尺度修建的楼梯，穿过门洞，并站到工作位置上；我不必先被判定为人，才有资格继续承担这里的任务。

我站在泵站内部，部件仍固定在背上。下一幕从一个看似很小的动作开始：我把它移到身前，伸手靠近设备。质量分布、惯量、角动量和接触条件随之改变，刚刚够用的双足模型将第一次面对操作与平衡同时发生。
