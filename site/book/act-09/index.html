第九幕

手伸向世界，力回到脚下——从传统全身控制到有边界的强化学习

全书路线

我已经走过八级台阶和窄门，在泵站内以稳定双脚支撑停下。替换部件仍固定在背部；现在要把它移到身前，用双手送进安装口。

这具人形身体在上一幕已经有双臂和双手，但它们只做过低作用的接口检查、保持与停止。双手承重、闭链配合、灵巧接触和精细安装还没有成立。手一伸出，身体的质量分布、角动量和脚下作用就已经改变；真正碰到设备以后，世界施加在手上的作用还会沿整具身体回到双脚。

因此，本幕不会用强化学习跳过传统控制。前十三节先完整建立质心动量、多接触、分层二次规划、质心模型预测控制、主动站位、双臂运动规划、闭链承重、灵巧接触、扰动恢复、可生存性和人员边界，再让质量、惯量、零位、摩擦、载荷和地面的细小偏差逐层暴露反复重建的代价。只有问题积累到“能否从反复变化中形成修正”以后，博士才命名强化学习。

后十节从一次尝试的边界开始，依次建立马尔可夫与部分可观测决策过程、Bellman 关系、策略梯度、演员—评论家、PPO、SAC、约束目标、课程、域随机化、模仿复用、残差、离线学习、分布外检测与运行时保障。学习策略始终只提出有边界的残差；状态估计、模型约束、传统控制、人员明确答复、安全屏障、后备控制、保护停止和急停不会被奖励分数取代。

手还没有碰到设备，脚下为什么已经变化

  1. 手还没有碰到设备，脚下为什么先变了
  2. 手推向设备的力，为什么会回到脚底
  3. 所有动作都重要时，哪一个应该先让步
  4. 这一刻没有跌倒，下一秒还来得及收回来吗
  5. 手能够到，为什么我还要先挪动双脚
  6. 两只手各自都有路，为什么一起走就堵住了
  7. 两只手都抓住部件，为什么还会互相较劲
  8. 部件已经对准，为什么指尖的一点滑动仍会毁掉安装

手部计划先进入质心动量和完整身体模型；真正施力后，手脚的接触力与力矩合量又必须同时满足单边、摩擦、压力中心、关节与功率边界。任务冲突引出严格层级，未来可恢复性引出模型预测控制，腕部奇异和脚下余量共同迫使我先换站位。左右手各自有路以后，独立规划、优先级、协调空间与约束流形先回答“两只手怎样共同到达”；双手抓住同一物体以后，闭链和内部作用再解释“物体没动，两只手仍在彼此用力”。指尖滑动则把问题推进到触觉、接触模式与手内调整。

每项偏差都能修，为什么整套方法仍越来越难维护

  1. 我知道自己受到了一股力，为什么还不知道它从哪里来
  2. 还能迈出一步，不等于还有安全的下一步
  3. 为了救回自己迈出的那一步，会不会伤到别人
  4. 只偏了一点点，为什么整套模型都要重新核对
  5. 每一次都能修好，为什么我仍觉得方法出了问题

相同残差可能来自地面、部件、传感或外部碰触，扰动观测不能凭一个数值给来源定案。恢复也不能只问“下一步够不够得到”，还要看后续是否仍有保持、放件和停止的路；人员进入以后，可用恢复空间还会改变。

传统方法没有被故意写弱。六类偏差先逐项完成测量、模型更新、依赖传播和留出复核；组合偏差还给出未经重调、局部修补与完整重建三种公平对照。问题不在于传统方法答不出，而在于身体与世界反复变化时，整条模型链需要一次次重新确认。

一次经验怎样改变下一次选择

  1. 我的一次尝试，究竟从哪里开始到哪里结束
  2. 一个动作眼前有利，怎样判断它对之后仍有价值
  3. 如果只看结果，我怎样知道哪一步该负责
  4. 同一批经验，为什么不能让我一下改得太多
  5. 旧经验怎样反复使用，又不把选择过早学窄
  6. 奖励越高，为什么行为反而可能越危险

同一传感画面可能对应不同隐藏摩擦和载荷，因此环境状态、执行观测、有限历史与 belief 必须分开；策略提出、保护链接受和身体实际施加的动作也不能混写。回报与 Bellman 关系让未来结果进入当前判断，policy gradient 和 Actor–Critic 再回答多步动作怎样分担结果。

PPO 只限制一次基于旧数据的更新，不限制物理作用；SAC 可以复用旧经验并保留动作分布，却不能把数据空白处的高价值估计当成事实。高奖励还可能来自激进伸手、永不移动或过力安装。课程与约束马尔可夫决策过程帮助整理任务，但期望成本不能保证每一时刻的硬边界，人员和急停更不能被总分抵消。

学来的动作在哪里停下，机械完成又在哪里结束

  1. 我能不能先在许多可能的身体里练习
  2. 平均表现不错，为什么最坏的几次更值得担心
  3. 学会的动作，什么时候必须把决定还给传统控制
  4. 部件终于装上了，为什么泵站仍没有启动

域随机化描述有来源、彼此相关并且物理可实现的一族变化，不是把所有参数无限拉宽。工坊里的示范可以作为初始化或先验，传统控制可以提供基线，学习只修正逐项写明单位、幅度和速率的残差。冻结留出、离线数据支持、OOD 漏报、低分位数与最坏组合决定候选能否在某个范围内继续被考虑；一次平均提升或一次留出成功都不等于开放世界泛化。

运行时真正拥有动作的不是最高奖励或最高价值估计。状态、人员答复、模型约束、截止时间、安全屏障和后备动作共同决定学习修正、传统控制、保持、放件、保护停止与急停之间的单向降级。最后的机械安装只有在对准、插入、锁定、卸载、双手退出和人员确认全部成立时才结束。
