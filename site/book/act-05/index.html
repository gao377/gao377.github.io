第五幕

手留下的尺度——从轮足身体到接触以前

全书路线

我把泵站维修毛坯送进旧工坊，却只能停在工作台旁看着吊架替我卸货。于是博士没有换掉原来的我，而是在同一具轮足身体上增加操作臂、安装座、张开夹爪、腕部力觉和视觉部件。毛坯由吊架承托以后不再算进身体质量；新部件加入以后，我成为一具 54 kg 的轮足移动操作身体。

多出一条手臂，不等于已经拥有一只可靠的手。关节状态、驱动与传动要先能互相解释，电脑里的身体也要和现在的我一致；坐标、时间和标定关系建立以后，我才可以从关节角算出手在哪里，再反过来寻找一种可能的姿态。

即使一个姿态能够到达，也不代表通往那里的一整段动作无碰、及时或仍属于当前世界。工具架会挡住肘部，第一条绕行路线可能别扭，老师傅移动一把工具也会使旧计划立刻过期。因此这一幕始终把整具身体、完整路径、当前场景和停止条件放在一起检查。

一条手臂怎样成为身体的一部分

  1. 东西已经送到，我为什么仍不能把它放下
  2. 装上手臂以后，它为什么还不是我的手
  3. 电脑里的身体，真和现在的我一样吗
  4. 同一个位置，为什么每个部件说得都不一样

这里先分清身体升级、部件在线和操作能力。七关节结构、驱动、传动、编码、质量与惯量、碰撞外形、坐标系图、时间和初始标定共同形成一具只在声明用途内可信的数字身体。

手怎样找到一个可能的姿态

  1. 关节依次转动，手会走到哪里
  2. 手只挪一点，为什么所有关节都要动
  3. 目标就在眼前，关节究竟该怎么转
  4. 明明还能动，为什么有些方向像被锁住了
  5. 我按模型伸到那里，为什么总会差一点

正运动学、雅可比矩阵、数值逆运动学、奇异值分解、冗余与标定并不是彼此分开的数学课。它们依次回答同一次伸手中的五个问题：手会在哪里、关节的小变化会造成什么、怎样反求姿态、哪些方向正在退化，以及为什么现实总比模型差一点。

一个姿态怎样成为当前世界中的无碰接近

  1. 一串姿势，为什么还不是一段动作
  2. 模型说没有碰撞，为什么手肘仍擦到工具架
  3. 工具架挡在中间，我该从哪边绕过去
  4. 最先找到的路，为什么走起来又慢又别扭
  5. 老师傅挪动一把工具，旧计划为什么立刻失效
  6. 手已经来到毛坯旁边，我为什么仍不能抓住它

同一组起终点可以形成 PTP、LIN 或 CIRC 等不同几何承诺；路径还要配上梯形速度、S 曲线或受约束的时间参数化，才成为带时间的轨迹。随后，离散节点无碰还不够，节点之间扫过的整段空间也要检查。采样规划帮助我找到不同绕法，轨迹优化改善其中一条，执行监测则在身体、状态或世界改变时撤销旧结论。

这一幕最终只形成张开夹爪的接触前接近方案。我已经能不碰到工具架地来到毛坯面前，却没有闭合夹爪：第一处接触在哪里、世界会怎样推回我的手、摩擦能否阻止滑落，以及怎样判断抓取真正成功，仍然是尚未回答的问题。
