第十二幕

我们怎样成为“我们”——从共同任务到相互拒绝与修复

全书路线

暴雨前的联合演练里，我、排水渠巡检设备和城市保障运输设备都完成了各自收到的命令，值班员与维修班组也各自守着职责。可三条绿色记录拼在一起，既没有覆盖真正的堵点，也没有等到清场答复。上一幕留下的共享世界、车队候选和本地经验，让我们能够交换信息；它们还不能回答“我们究竟在共同完成什么”。

我先从这次失败追问：如果每个人都没有单独做错，为什么共同结果仍然失败？博士由此引入共同任务、任务分配、图与共识、协同操作、多主体学习和韧性协同。理论不是另一条独立路线：我会在新问题里想起过去用过的匹配、图、MPC、抓取矩阵和强化学习，再重新核对当前的身体、权限、通信与他者答复，确认旧方法究竟能复用到哪里。

贯穿本幕的是五位持续参与者。人形的我 H 继承四足、轮足、轮足移动操作身体直到人形的完整历史；低矮防水的排水渠巡检设备 D 和道路尺度的城市保障运输设备 T 保留各自身体；值班员 O 与维修班组 M 保留彼此不能代替的知识和权限。终端、地图服务和传感节点可以保存或传递信息，却不会因为出现在通信图中就自动成为承诺的主体。

共同计划：先弄清我们在做什么、由谁来做

  1. 每个人都完成了自己的任务，为什么演练还是失败了
  2. 谁去做哪件事，怎样不靠一句“你最合适”
  3. 两件事必须同时发生，分配表为什么还不够
  4. 每个人只说自己的代价，怎样形成共同分配
  5. 搭档失联或拒绝以后，承诺怎样重新安排

三份局部成功记录先被改写成一个可检查的共同任务。匈牙利算法处理带硬禁配的一对一匹配，混合整数线性规划与时间网络补上共同就绪、资源和时窗，MURDOCH 与基于共识的任务束算法再说明局部报价怎样形成分布式分配。可任务执行中还会出现一种不能被故障标签抹去的回应：当排水渠设备的密封与健康有效性为 UNKNOWN 时，它拒绝进入低洼水道。旧入内任务随即取消，计划改为外部观测并请求维护；拒绝不是一句说明，而是共同计划真正发生了变化。

共同行动：信息相连以后，身体怎样不彼此拖垮

  1. 队形不是图案，它究竟替我们守住什么
  2. 我只听见邻居，怎样知道大家会不会走到一起
  3. 消息晚到或少发一次，队形为什么会改变
  4. 看起来形状没变，为什么结构仍可能塌掉
  5. 每个人只算自己的下一步，整体约束谁来守
  6. 两副身体抬着一个物体，为什么各自正确仍会互相用力

我从通信距离、通道宽度和共同搬运的约束出发，重新理解 Boids、Vicsek、图、拉普拉斯矩阵、共识、刚性和分布式模型预测控制。队形不再是漂亮的几何图案，共识也不等于真相；它们分别在声明的图条件、时延、新鲜度、触发规则和任务理由下工作。到了共同搬运，抓取矩阵把对象所需作用与身体施加的接触作用连起来，零空间里的内部力则解释了为什么对象几乎不动，两副身体仍可能互相用力。

共同学习与承担：搭档会变化，也可以拒绝我

  1. 人已经加入行动，模型能替他回答吗
  2. 搭档也在学习，为什么我的环境不再稳定
  3. 同一个共同回报，为什么会学出不同的配合
  4. 换了搭档仍能配合，究竟算泛化还是运气
  5. 如果错误来自伙伴，我们怎样继续相信彼此

共享自治先把人的可观察动作、模型推断、明确答复、权限决定和已执行动作分开；模型可以帮助提问，却不能替人答复。随后，分散可观测马尔可夫决策过程与“集中训练、分散执行”说明搭档也在学习时为什么会出现非平稳性；MADDPG、QMIX、IPPO、MAPPO、PRIMAL、模仿学习和学习式通信各自承担不同问题，并与 CBS、PIBT、ORCA 等经典方法保持同口径比较。所有学习结果仍处于影子或尚未运行状态，候选不取得安全关键控制权。

最后，故障诊断、韧性共识与拜占庭故障假设帮助我们区分异常报告、错误来源和允许过滤的条件，却不能凭一个极端值宣布谁有恶意。维修班组答复为 MISSING 时，当前预测器给出的 91% 不是通用阈值、法规、现实安全概率或人的答复。我拒绝启动，旧任务租约与排程被撤销，其余参与者转入等待明确答复、外部监测和暴雨准备。这是第二次方向相反的拒绝；它与排水渠设备的拒绝共同说明，成为“我们”并不要求任何一方先失去自己的边界。

十二件产物怎样接成一条证据链

参与者能力与边界表
→ 共同任务合同
→ 任务分配记录
→ 编队与通信方案
→ 协同关系图
→ 协同操作方案
→ 共享自治边界
→ 多智能体策略影子候选
→ 搭档变化评测方案
→ 韧性协同方案
→ 相互拒绝与修复合同
→ 协同证据包

这条链保存谁能做什么、共同成功怎样判断、分配和队形怎样修订、对象与人的边界怎样保留、学习候选怎样评估，以及错误和拒绝怎样触发修复。它不把三种身体平均成一个通用机器人，也不把值班员和维修班组压成一个人类接口。

返回第十一幕：档案说，“我”来过这里

暴雨真正到来时，我们已经能够提出共同任务、交换有边界的信息、共同搬运、拒绝错误行动并修复计划。终幕会把这些关系放进同一场城市危机：地图、通信、人员、设备和远端结果同时变得不可靠，而“我”也要继续追问——究竟是城市塑造了我的身体和行动，还是包括我在内的行动者一直在共同塑造这座城市。
