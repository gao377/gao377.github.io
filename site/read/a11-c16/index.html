第十一幕

传统定位看不清时，学习表示能补什么——学习状态估计、SLAM 与神经表示

反光水面和蒸汽让传统特征稀少，定位协方差迅速扩大。一个学习候选却在回放中给出稳定表示，轨迹也很平滑。

我先感到它“看懂了”走廊，随后发现训练集里包含同一路段的相邻片段。稳定究竟来自可迁移结构，还是来自记住背景，还没有答案。

“学习定位”其实包含不同接口

博士把几种常被混叫的方法分开：

  • learned feature：从图像产生可匹配特征/描述子，几何估计仍由传统模块完成；
  • learned odometry：从 image/IMU/proprioception 产生相对运动候选；
  • learning-enhanced SLAM：学习观测或因子进入图优化，地图/回环/一致性仍显式存在；
  • neural scene representation：用神经场或隐式表示编码场景，再支持跟踪/重建；
  • end-to-end localization：直接输出 pose/state proposal。

它们学习部分的 authority、可解释残差和 failure mode 不同，不能用一个“AI 定位”标签覆盖。

zk=fϕ(Ik−r:k, uk−r:kimu, qk−r:k, ck−r:k),(1)z_k=f_\phi(I_{k-r:k},\,u^{imu}_{k-r:k},\,q_{k-r:k},\,c_{k-r:k}), \tag{1}zk​=fϕ​(Ik−r:k​,uk−r:kimu​,qk−r:k​,ck−r:k​),(1)

输出 zkz_kzk​ 可以是 feature、相对位姿 ΔT^\Delta\hat TΔT^ 或状态 proposal x^kL\hat x_k^Lx^kL​，必须同时带 model/data/body/sensor/calibration revision、clock、uncertainty 和 OOD 状态。

学习候选仍要经过身体和几何

传统基线给出 x^kG\hat x_k^Gx^kG​ 及不确定度。学习候选不能因数值平滑直接替代它，要检查：

  • 相邻相对位姿与 IMU、关节/轮足里程和接触约束是否一致；
  • 投影、回环、地图和已知结构的几何残差；
  • body/sensor/calibration revision 是否属于训练/验证支持；
  • 当前位置、天气、蒸汽、反光和动态物体是否 OOD；
  • 长时 drift、失败恢复、latency 与 deadline；
  • 训练/holdout 是否含同一路段、同 raw 或近重复泄漏。

可将学习输出作为一个带协方差的候选因子，而非事实：

rL(x)=Log⁡ ⁣((TL)−1T(x)),JL=rLTΣL−1rL.(2)r_L(x)=\operatorname{Log}\!\left((T_L)^{-1}T(x)\right), \qquad J_L=r_L^{\mathsf T}\Sigma_L^{-1}r_L. \tag{2}rL​(x)=Log((TL​)−1T(x)),JL​=rLT​ΣL−1​rL​.(2)

这不保证 ΣL\Sigma_LΣL​ 已校准。只有在独立路线和冻结 OOD/残差协议下，才知道低 JLJ_LJL​ 是否有意义。

影子比较让它先学会“不同意”

冻结传统基线、地图、身体、标定和数据划分身份
让学习候选运行在相同回放组上，但不给它控制权
记录特征、相对运动或状态提议，以及不确定度和延迟
逐组比较几何、运动学、接触、漂移、尾部和分布外情况

如果出现谱系泄漏、版本错配、分布外或物理不一致：
    候选继续保持 SHADOW，并将该组写为 UNKNOWN 或 REJECTED
否则：
    保留与传统基线配对的影子证据，等待后续复核
不把学习提议写入当前状态，也不让它进入运动控制

低不确定的传统结果与学习候选冲突时，不按“谁更自信”投票；先查各自可观测假设、来源和残差。传统特征在蒸汽里可能退化，学习表示在新场景也可能过度自信。两者的 disagreement 本身进入 failure atlas。

初始的学习导航影子候选因此保存接口、训练与验证谱系、配对基线、分布外状态、几何与接触检查、漂移、尾部和延迟目标。它还没有任何现实性能记录。

练习

练习 1：传统不确定，学习候选残差低

传统定位因蒸汽给出高不确定度，学习候选在回放中残差低；训练集却含同一路段。判断能否采纳，以及首先要修什么证据。

练习 2：新相机 revision 没进训练

候选输入 shape 相同，新相机却改变曝光、镜头和 calibration。说明它应处于什么状态，shape 为什么不足。

练习 3：学习候选与接触运动矛盾

候选说身体向前移动，足底接触与关节运动学却支持近似静止。给出处置、可能解释和复核方法。

参考答案

学习候选可以建议“我在哪里”。下一段潮湿地面上，它又建议“可以走”，传统几何地图却把那里标成不可通行：位置之外，世界还在争论什么动作可供我采取。
