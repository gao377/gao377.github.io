给定接触集合时，判断流程为：

检查浮动机身模型、关节记录和声明的接触集合
→ 检查坐标系、行顺序、单位、配置版本和时间是否一致
→ 检查质量矩阵是否符合物理条件，接触雅可比矩阵是否满行秩
→ 按固定符号组装 KKT 系统
→ 用线性求解得到候选整机加速度与接触作用，不显式形成逆矩阵
→ 从原方程独立复算动力学残差 r_d 与接触残差 r_c
