第七幕：城市不会停下来等我

  1. 序幕
  2. 第一幕 /我能动，却还站不起来
    1. 为什么命令发出以后，身体却开始发抖——下一次电机该怎样用力

    3. 关节都转对了，脚为什么还是踩偏——角度怎样一路走到脚尖
    4. 关节只动一点，脚为什么会改变方向——从当前位置看身体能往哪里走
  3. 第二幕 /四条腿都听话，为什么我仍然会倒下

    4. 传感器都在线，我究竟以什么姿态站着——几份局部观测怎样拼成同一个现在
    5. 博士什么时候才肯撤走维护架——一次没有倒下为什么还不够
  4. 第三幕 /碎石路不会等我学会

    3. 眼前有石头，为什么不能踩下去——落脚位置必须同时适合地面和身体
    4. 三只脚都在地上，谁在托住正在移动的我——从质心与动量看整体受力
    5. 脚、机身和关节都想做不同的事，谁先让步——优化怎样把它们接成全身控制
    6. 每一步都能算通，为什么前面却走不通——预测控制怎样提前看见后果
    7. 计划说脚已经落稳，我为什么还在滑——意外接触怎样触发恢复
    8. 穿过通道以后，哪些地方仍不能说安全——一次四足行走怎样留下可复核的记录
  5. 第四幕 /我学会滚动，也知道何时迈步

    2. 身体换了一部分，旧经验还能用吗——轮足本体重建与标定
    3. 轮子为什么只愿意沿一个方向走——滚动与非完整约束
    4. 我数出的距离可信吗——轮式运动学与直线行程
    5. 路向左弯，为什么四个轮子在互相拉扯——固定轮转弯与轮式雅可比
    6. 轮子说我到了，墙角为什么还没到——里程计误差与协方差
    7. 斜路为什么把货物推向一边——载荷转移与滚动全身控制
    8. 一根矮管为什么能拦住我——轮足混合运动与预测控制
    9. 轮子越转越快，我为什么反而没有前进——滑移与摩擦边界
    10. 地面还能借给我多少力——摩擦估计与牵引控制
    11. 继续滚、抬腿，还是停下来——能耗、风险与模式决策
    12. 我已经到了，为什么修理仍未开始——路线复盘与操作缺口
  6. 第五幕 /我学会把手伸到正确的位置
    1. 东西已经送到，我为什么仍不能把它放下——操作缺口与身体升级
    2. 装上手臂以后，它为什么还不是我的手——结构、驱动与操作身体状态

    4. 同一个位置，为什么每个部件说得都不一样——坐标图与初始标定
    5. 关节依次转动，手会走到哪里——机械臂正运动学
    6. 手只挪一点，为什么所有关节都要动——机械臂雅可比矩阵与局部映射

    9. 我按模型伸到那里，为什么总会差一点——优化标定与误差传播
    10. 一串姿势，为什么还不是一段动作——路径、轨迹与时间参数化
    11. 模型说没有碰撞，为什么手肘仍擦到工具架——配置空间与碰撞距离
    12. 工具架挡在中间，我该从哪边绕过去——采样式运动规划
    13. 最先找到的路，为什么走起来又慢又别扭——轨迹优化与混合规划
    14. 老师傅挪动一把工具，旧计划为什么立刻失效——执行监测与在线重规划
    15. 手已经来到毛坯旁边，我为什么仍不能抓住它——无碰接近与接触边界
  7. 第六幕 /我学会在接触中完成一件事
    1. 世界推回我的手时，究竟是谁在使力——第一次接触与机械臂动力学
    2. 同一个动作换了负载，为什么手臂立刻变了——递归动力学与参数辨识
    3. 关节都跟上了，为什么手还是撞得太重——跟踪与操作空间控制
    4. 我想轻轻压住它，怎样知道自己用了多大力——力控制与混合控制
    5. 同样的力碰到不同材料，为什么结果完全不同——阻抗、导纳与接触稳定
    6. 眼睛已经看准了，为什么手一碰位置又变了——RGB-D 与操作感知误差链

    10. 哪两个地方值得夹，怎样比较它们——闭合、抓取质量与候选筛选
    11. 抓住一次以后，我凭什么相信它没有滑——抓取执行、验证与重新抓取
    12. 毛坯进了夹具，为什么最后几毫米最难——接触序列、夹具与柔顺插入
    13. 路线没有撞到东西，为什么一推下去还是偏了——动力学与接触规划
    14. 一步做对了，为什么整件零件仍会失败——任务编排、恢复与成功谓词
    15. 老师傅每次动作都不同，为什么结果却一样——遥操作、示范与动作语义
    16. 一段经历从哪里开始，又该在哪里结束——任务经历与数据合同
    17. 照着成功动作学，为什么离开示范就越错越远——行为克隆、分布偏移与数据集聚合
    18. 一个动作还没做完，为什么下一次推理已经改口——动作分块与 ACT
    19. 同一个局面有几种好做法，我该怎样保留这种差别——扩散策略、流匹配与工艺闭环
  8. 第七幕 /城市不会停下来等我
    1. 地图上有起点，我为什么仍不知道自己在哪里——导航任务、身体状态与观测责任
    2. 传感器都在工作，为什么走出的路对不上——测量物理与时空标定
    3. 轮子说我走了这么远，身体真的走了吗——轮足惯性状态估计与误差状态滤波
    4. 两张照片之间，究竟是什么发生了变化——视觉里程计与几何退化

    18. 旧路被封，我该不该走进未知侧巷——主动探索、信息增益与安全返回

    26. 没有人统一指挥，怎样不把彼此永远困住——PIBT、终身协调与 ORCA 的边界
  9. 第八幕 /我重新组织身体，走过台阶和窄门
