同一个动作换了负载，为什么手臂立刻变了——递归动力学与参数辨识

上一章结束后，毛坯仍由夹具台和吊架承托。我把手退回无接触位置，重复一段短轨迹：先空着夹爪运动，再由吊架把一小部分重量缓慢转移到夹爪。转移量只有 30 N30\ \mathrm N30 N，远不是让我独立举起整块 10 kg10\ \mathrm{kg}10 kg 毛坯。

两次轨迹的关节角几乎重合，可肩、肘和腕部记录的作用变了，末端也出现新的偏差。我最先想到的是：“既然重量加在手上，把它补到最后一个关节不就行了吗？”

博士让我先把手臂想成一串相互传话的连杆。末端多出的作用不会只停在腕部；它沿关节链传回底座，每个关节承担多少，取决于当时的姿态、运动和力臂。博士把“已知运动，求各关节需要给出多大作用”称为逆动力学，又把沿机构逐段计算的方法称为递归刚体动力学。

三个相似的算法其实在回答三种问题

上一章的模型可简写为

M(q)q¨+h(q,q˙)=τ+Jc(q)TFext,M(q)\ddot q+h(q,\dot q) =\tau+J_c(q)^{\mathsf T}F_{\mathrm{ext}},M(q)q¨​+h(q,q˙​)=τ+Jc​(q)TFext​,

其中

h=C(q,q˙)q˙+g(q)+τf(q,q˙).h=C(q,\dot q)\dot q+g(q)+\tau_f(q,\dot q).h=C(q,q˙​)q˙​+g(q)+τf​(q,q˙​).

同一等式可以从不同方向求解。

已知运动，求关节作用：RNEA

若 qqq、q˙\dot qq˙​、q¨\ddot qq¨​ 和外部作用已知，需要计算 τ\tauτ，可写成

τ=M(q)q¨+h(q,q˙)−JcTFext.\tau=M(q)\ddot q+h(q,\dot q)-J_c^{\mathsf T}F_{\mathrm{ext}}.τ=M(q)q¨​+h(q,q˙​)−JcT​Fext​.

递归 Newton–Euler 算法通常简称 RNEA。它先从底座向末端递推各连杆的速度和加速度，再从末端向底座递推惯性作用与外部作用，最后得到关节作用。对树形刚体系统，它的工作量随连杆数近似线性增长。

已知姿态，求整张质量矩阵：CRBA

若控制器需要显式使用 M(q)M(q)M(q)，可以逐列调用逆动力学，但会重复许多计算。复合刚体算法，简称 CRBA，先把每个子树的惯量向父节点合并，再组装质量矩阵。对一般串联链，完整矩阵有 n2n^2n2 个元素，因此计算和存储至少要面对二次规模。

质量矩阵不是“七个关节各自的质量表”。非对角项描述一个关节加速时怎样牵动其他关节；末端负载会改变整张矩阵，而不只改变最后一行。

已知关节作用，求运动变化：ABA

若已知 qqq、q˙\dot qq˙​、τ\tauτ 和外部作用，要预测 q¨\ddot qq¨​，形式上是

q¨=M(q)−1(τ+JcTFext−h).\ddot q=M(q)^{-1} \left(\tau+J_c^{\mathsf T}F_{\mathrm{ext}}-h\right).q¨​=M(q)−1(τ+JcT​Fext​−h).

直接形成逆矩阵既没有必要，也会隐藏数值问题。铰接体算法，简称 ABA，通过向内、向外的递推直接求前向动力学；对常见树形系统同样可做到近似线性规模。

三者不能只靠名字相互验证。对同一状态，可以做一轮闭环复算：

  1. 用 RNEA 从 q,q˙,q¨q,\dot q,\ddot qq,q˙​,q¨​ 得到 τ\tauτ；
  2. 用 CRBA 得到 MMM，另算 hhh，检查两种逆动力学结果；
  3. 把同一 τ\tauτ 交给 ABA，检查是否还原原来的 q¨\ddot qq¨​；
  4. 再检查单位、外力符号、关节顺序和功率。

数值接近只说明实现彼此自洽。若三种实现共用一份错误惯量或坐标，它们仍可能一起算错。

三十牛顿怎样传回七个关节

在足够慢、加速度可忽略的受控负载转移中，两次关节作用之差近似满足

Δτ≈Δgpayload(q)−Jc(q)TΔFext+Δτf.\Delta\tau \approx \Delta g_{\mathrm{payload}}(q) -J_c(q)^{\mathsf T}\Delta F_{\mathrm{ext}} +\Delta\tau_f.Δτ≈Δgpayload​(q)−Jc​(q)TΔFext​+Δτf​.

到底把 30 N30\ \mathrm N30 N 记作工具负载还是腕部以下的外部作用，取决于传感器安装位置和系统边界。二者不能同时完整计入，否则同一重量会被算两次。

假设当前姿态下，声明方向的 30 N30\ \mathrm N30 N 转移经雅可比矩阵映射后，模型预计七个关节变化为

Δτpred=[0.0−6.3−3.00.20.0−0.10.0]T N m.\Delta\tau_{\mathrm{pred}} = \begin{bmatrix} 0.0&-6.3&-3.0&0.2&0.0&-0.1&0.0 \end{bmatrix}^{\mathsf T}\ \mathrm{N\,m}.Δτpred​=[0.0​−6.3​−3.0​0.2​0.0​−0.1​0.0​]T Nm.

屏幕记录若为

Δτmeas=[0.1−6.2−3.10.20.0−0.10.0]T N m,\Delta\tau_{\mathrm{meas}} = \begin{bmatrix} 0.1&-6.2&-3.1&0.2&0.0&-0.1&0.0 \end{bmatrix}^{\mathsf T}\ \mathrm{N\,m},Δτmeas​=[0.1​−6.2​−3.1​0.2​0.0​−0.1​0.0​]T Nm,

两者在这一姿态下相近，只支持“负载传播方向与量级暂时一致”。它不证明工具质量、摩擦和关节零位都正确，更不能把 30 N30\ \mathrm N30 N 的局部检查外推到整块毛坯。

残差不是某个参数的姓名

把模型预测与测量之差记为

r=τmeas−τmodel.r=\tau_{\mathrm{meas}}-\tau_{\mathrm{model}}.r=τmeas​−τmodel​.

同样大小的残差可能来自不同机制：

  • 随姿态和重力方向变化，可能与连杆或工具惯量有关；
  • 随速度正负翻转，可能与 Coulomb 摩擦有关；
  • 随速度大小平滑增长，可能与黏性摩擦或传动损耗有关；
  • 静止时近似恒定，可能是关节作用偏置、预载或零位误差；
  • 只出现在腕部记录中，可能是力觉偏置、工具重力补偿或坐标错误；
  • 在快速运动中随时间错开，可能来自同步误差或加速度估计。

我先看见的是这些“纹理”，博士才引入动力学参数辨识：用多组受控运动与作用记录，估计能解释系统动力学的参数，而不是拿一条残差曲线随意改一个数。

为什么动力学对参数可以写成线性的

刚体动力学对关节状态是非线性的，但对质量、质心一阶矩和惯量张量等适当组合可以整理为

Y(q,q˙,q¨)π=τ−JcTFext.Y(q,\dot q,\ddot q)\pi = \tau-J_c^{\mathsf T}F_{\mathrm{ext}}.Y(q,q˙​,q¨​)π=τ−JcT​Fext​.

YYY 是由已知状态构成的回归矩阵，π\piπ 是待估参数组合。把多时刻叠起来，得到

Yπ=T.\mathcal Y\pi=\mathcal T.Yπ=T.

若还要估计摩擦，可在回归矩阵中加入 sgn⁡(q˙)\operatorname{sgn}(\dot q)sgn(q˙​)、q˙\dot qq˙​ 等列，但零速附近的摩擦并不服从简单符号函数，必须声明剔除或另建模型。

并非每个连杆参数都能单独从关节作用里看见。有些参数只以和或差出现；有些列在当前轨迹中几乎重合。对缩放后的 Y\mathcal YY 做奇异值分解，若某些奇异值远小于阈值，就只能估计基参数或可辨组合，不能给所有原始参数各填一个看似精确的小数。

列尺度也很重要。质量、惯量和摩擦的单位不同，不先缩放便直接比较奇异值，可能把单位差异误当成可辨识程度。参数协方差同样依赖噪声模型、相关性和模型假设；很小的拟合方差不能消除结构性偏差。

一套不会把拟合当成身体事实的流程

输入：
  当前几何、惯量、摩擦、工具和传感器版本
  受控空载与 30 N 负载转移记录
  关节、速度、作用、腕部记录和采样时刻

计算一致性：
  1. 对同一状态运行 RNEA、CRBA 复算与 ABA 回代
  2. 检查外部作用是否只在模型的一侧计入一次
  3. 检查功率、单位、关节顺序和时间对齐

建立回归：
  4. 选择不越过限位、速度、作用和支撑边界的激励轨迹
  5. 把用于辨识的记录与用于验证的姿态分开
  6. 构造并缩放回归矩阵，检查秩、条件和相关方向
  7. 估计可辨参数、残差、协方差和适用负载范围

发布决定：
  8. 若独立姿态也改进，登记新候选版本
  9. 若只改善拟合集、参数不可辨或物理量不合理，保留旧版本
  10. 若同步、支撑或传感器身份不明，停止解释

辨识轨迹和复核轨迹分开，是为了防止模型只记住用于估计的动作。若新参数让拟合集残差下降，却让未参与估计的姿态变差，我不会把它写成新的身体事实。

Roy Featherstone 的空间向量与刚体动力学资料提供了 RNEA、CRBA 和 ABA 的原始算法实现与说明；Gautier 与 Khalil 关于基惯性参数激励轨迹的论文讨论了回归尺度和条件。本章只采用与当前七关节、固定底盘、受控负载切片一致的部分。

回到同一段动作

我不再说“负载把最后一个关节压重了”。更准确的说法是：负载改变了整条关节链的动力学；记录中的差异还要经过算法交叉检查、残差归因和独立姿态复核，才能成为一个参数更新候选。

模型现在能够更快地回答“为了走出这段运动，各关节应给出什么作用”。可下一次接近固定基准时，所有关节都跟得很准，腕部作用仍突然升高。准确跟踪关节，并没有自动带来温和的手。

练习

四题沿用同一次锁定底盘复试。毛坯始终由夹具和吊架承托；所谓负载变化只指声明的 30 N30\ \mathrm N30 N 转移。

练习 1：三十牛顿转移以后，差值是否足以更新工具质量

同一静止姿态下，模型预计关节作用变化为

[0.0,−6.3,−3.0,0.2,0.0,−0.1,0.0]T N m,[0.0,-6.3,-3.0,0.2,0.0,-0.1,0.0]^{\mathsf T}\ \mathrm{N\,m},[0.0,−6.3,−3.0,0.2,0.0,−0.1,0.0]T Nm,

测得变化为

[0.1,−6.2,−3.1,0.2,0.0,−0.1,0.0]T N m.[0.1,-6.2,-3.1,0.2,0.0,-0.1,0.0]^{\mathsf T}\ \mathrm{N\,m}.[0.1,−6.2,−3.1,0.2,0.0,−0.1,0.0]T Nm.

判断我能否据此直接更新工具质量。说明下一项检查怎样区分“负载传播大体正确”与“工具参数已经准确”。

练习 2：残差随着运动方向翻转

卸去受控负载后，同一姿态附近以相同速率正、反向运动。第三关节残差分别为 +0.25 N m+0.25\ \mathrm{N\,m}+0.25 Nm 和 −0.24 N m-0.24\ \mathrm{N\,m}−0.24 Nm；停住后残差回到 0.03 N m0.03\ \mathrm{N\,m}0.03 Nm。有人建议增大第三连杆质量。

判断更应先检查哪种机制，应增加什么记录，并说明错误增大质量会怎样影响别的姿态。

练习 3：腕部和关节侧只有一边出现固定偏差

工具静止且明确无接触。腕部法向记录在重新校偏前为 3.2 N3.2\ \mathrm N3.2 N，校偏后接近零；关节作用残差在两次校偏前后没有变化。判断这组现象更支持力觉偏置还是关节动力学参数错误，以及我能否用腕部校偏修正关节模型。

练习 4：拟合很好，但有一个方向几乎看不见

缩放后的回归矩阵有四个候选参数方向，奇异值为

[18.0, 3.7, 0.8, 0.006],[18.0,\ 3.7,\ 0.8,\ 0.006],[18.0, 3.7, 0.8, 0.006],

当前判别阈值为 0.020.020.02。新参数让辨识记录的均方残差下降一半，却让独立姿态残差增加 35%35\%35%。决定我应发布新参数、只发布前三个原始参数、补充激励还是回退旧模型，并说明原因。
