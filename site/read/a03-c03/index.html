博士让我先看自己的速度。如果身体正在向前移动，脚落下时，髋部已经不在生成曲线那一刻的位置。石面即使足够平，也可能离那时的髋部太远；位置即使够近，也可能太高、太斜，或者只是一次已经过期的测量。

我原先把落脚点当成地面上的一个属性，仿佛石头会自己回答“能踩”或“不能踩”。现在问题变了：同一块石头，对不同速度、不同腿长、不同测量时刻的身体，答案并不相同。

博士这才把“根据身体将要到达的位置，先提出一个足端目标，再让地形和身体共同筛选它”的过程称为落脚选择。它不是从许多点中挑一个看起来最漂亮的点，而是保存一条可以复查的因果链：

这份地面记录是否仍可信
→ 身体按当前速度还会前进多少
→ 原始目标落在哪里
→ 腿是否够得到
→ 地面是否太斜或太高
→ 测量误差是否吃掉了剩余余量
→ 保留、改选，还是停下

  • 另外三只支撑脚需要提供多大的作用；
  • 这些作用是否落在摩擦边界内；
  • 机身会怎样平移和转动；
  • 完整摆动路径是否与三维地形碰撞；
  • 关节和执行器是否能按时实现；
  • 触地以后地面会不会移动。
